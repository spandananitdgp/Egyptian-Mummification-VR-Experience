Mummification in VR

Experience Time Travel To The 2600 B.C Egyptian Mummification Era

View on GitHub Download .zip Download .tar.gz

Description

This virtual reality experience leverages the telepresence and takes the user back to about 2600 B.C, and lets user be a part of the Egyptian mummification process. This immersive VR experience can be deployed to Google Cardboard or Oculus Rift. It is developed using Unity 3D Gaming Engine. This project is being developed as a class project for the Virtual Reality (CIS6930) class at University of Florida.

Egyptian Mummification VR Experience
